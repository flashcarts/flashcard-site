Download BL2CK 1.31.0 (Amaze3DS)

Description

The Amaze3DS line of carts are based on R4iTT’s Acekard-based carts. Though it is slightly different in that it boots from a specific memory address (0x02000000), where as normal R4iTT carts don’t really care about the address which they boot from.

Setup Instructions (BL2CK)

  1. Format the SD card you are using to FAT32
  2. Download BL2CK 1.31.0 Amaze3DS
  3. Extract the BL2CK 1.31.0 Amaze3DS .zip file to your SD card root.
  4. Place any .nds game ROMs you’d like to play into a folder of your choice which isn’t the __rpg folder.
    • (Optional) You may want to create a Games folder to store your ROMs in to keep the root of your SD card organised.
Cheat database location(s):
